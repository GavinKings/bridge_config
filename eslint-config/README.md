# Eslint Config of gavinkings

## Install

```shell
# with npm
npm i @gavinkings/eslint-config -D

# with yarn
yarn add @gavinkings/eslint-config -D

# with pnpm
pnpm add @gavinkings/eslint-config -D
```

## Usage

```js
  "eslintConfig": {
    "root": true,
        "extends": [
        "@gavinkings"
    ]
}
```
