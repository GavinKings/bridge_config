# Stylelint Config of gavinkings

## Install

```shell
# with npm
npm i @gavinkings/stylelint-config -D

# with yarn
yarn add @gavinkings/stylelint-config -D

# with pnpm
pnpm add @gavinkings/stylelint-config -D
```

## Usage

```js
  "stylelint": {
    "extends": [
        "@gavinkings/stylelint-config"
    ]
}
```
